
🦅 Falcon Panel (پنل عقاب).

✨ ویژگی‌های اصلی
داشبورد کلی: مشاهدهٔ آمار لحظه‌ای کاربران، فعالیت‌ها و وضعیت سرور.

مدیریت کاربران: جستجو، فیلتر، ویرایش و تغییر سطح دسترسی کاربران (فعال/غیرفعال).

سیستم احراز هویت قوی: لاگین امن با JWT و مدیریت نشست‌ها (Session).

تاریخچهٔ عملیات (Activity Log): ثبت لاگِ ورود، خروج و تغییرات حساس برای پیگیری.

تنظیمات پویا: امکان تغییر برخی تنظیمات سیستمی بدون نیاز به ریستارت.

پوستهٔ واکنش‌گرا: هم روی لب‌تاپ و هم موبایل، نمایش مناسبی داره (حالت شب/روز).

🛠 تکنولوژی‌ها
Backend: Python (Django Rest Framework)

Frontend: React.js + TailwindCSS

Database: PostgreSQL

Cache: Redis (برای کش کردن درخواست‌های تکراری)

Deployment: Docker + Nginx
